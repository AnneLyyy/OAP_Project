# ІНСТРУКЦІЯ ДО ПРОЄКТУ

## 1. ІНСТРУКЦІЯ ЗІ ВСТАНОВЛЕННЯ ТА ЗАПУСКУ
Для коректної роботи проєкту необхідно мати встановлений Node.js.

Крок 1: Встановлення необхідних модулів та залежностей.
Виконайте в терміналі:
> npm install

Крок 2: Запуск сервера у режимі розробки.
Виконайте в терміналі:
> npm run dev

Адреса локального сервера за замовчуванням:
http://localhost:3000

## 2. СПИСОК ДОСТУПНИХ ЕНДПОІНТІВ (API ENDPOINTS)

### Керування подіями:
- GET    /api/events     : Отримати повний список усіх подій.
- POST   /api/events     : Додати нову подію (реєстрація користувача).
- PUT    /api/events/:id : Оновити інформацію про подію за її ідентифікатором.
- DELETE /api/events/:id : Видалити подію з бази даних за її ідентифікатором.

### Сервісні запити:
- GET    /health         : Перевірка працездатності сервера.

## 3. ТЕХНІЧНІ ХАРАКТЕРИСТИКИ
- Платформа: Node.js
- Основний фреймворк: Express
- Формат обміну даними: JSON
